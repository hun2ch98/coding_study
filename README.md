<h3> 🌈 Convention Rule </h3>

<b> Issue </b> | `[플랫폼] 문제이름`
<br>ex) [Programmers - 고득점 Kit] 포켓몬

<b> PR </b> | `[스터디 날짜] 이름 #이슈번호`
<br>ex) [2023-01-25] dain

<b>Commit </b> | `커밋내용 문제이름 파일명`
<br>ex) Create 포켓몬 dain.py
<br>ex) Update README.md
